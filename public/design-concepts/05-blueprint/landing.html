데이터는 사용자 소유

모든 주요 엔티티는 사용자 소유권에 묶입니다. API 라우트는 클라이언트가 보낸 사용자 ID를 신뢰하지 않고, 보안 세션 쿠키에서 사용자를 도출해 소유권을 검증합니다.

  • 🔐서버 전용 모델 호출 — 모든 AI 요청은 서버 측에서만 실행됩니다.
  • 🔑암호화 자격증명 — 사용자별 공급자 키는 DB 암호화로 저장됩니다.
  • 🍪HttpOnly 세션 — 보안 쿠키 기반 이메일/비밀번호·Google 인증.

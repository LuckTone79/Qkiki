# Qkiki UI 디자인 컨셉 5종 (예시용 목업)

> **중요:** 이 폴더는 **디자인 탐색용 예시 목업**입니다. 실제 제품 코드(`src/`)에는
> 아무것도 적용되지 않았으며, 제품의 기능/구성/라우팅은 그대로입니다.
> 사용자가 "적용하라"고 지시하기 전까지는 시안으로만 둡니다.

## 무엇을 만들었나

현재 Qkiki의 **성격과 구성은 그대로 유지**하면서 **UI(디자인) 측면만** 5가지 방향으로
재해석했습니다. 각 컨셉은 다음 두 화면을 포함합니다.

- **랜딩페이지** (`landing.html`) — 비로그인 방문자용 마케팅 페이지
- **세부페이지 / 워크벤치** (`workbench.html`) — 로그인 후 핵심 작업 화면

제품의 핵심 정체성(여러 AI 모델 → 병렬 비교 → 순차 검토 체인 → 결과 분기 →
프로젝트/프리셋/파일첨부)은 모든 시안에 동일하게 반영되어 있습니다.

## 보는 법

`design-concepts/index.html` 을 브라우저로 열면 5개 시안을 한눈에 모아 비교하고
각 랜딩/워크벤치로 이동할 수 있습니다. (Tailwind Play CDN을 사용하므로 인터넷 연결 필요)

## 5가지 컨셉

| # | 이름 | 무드 | 핵심 특징 | 핵심 색 |
|---|------|------|-----------|---------|
| 01 | **Aurora** | Apple 홈페이지 스타일 | 초대형 타이포, 넉넉한 여백, 중앙 정렬 히어로, 스크롤 페이드인, 떠 있는 제품 프리뷰 | 화이트/그레이 + 틸·블루·바이올렛 그라데이션 |
| 02 | **Studio** | 에디토리얼 미니멀 | 현재 크림·틸 색감 계승·세련화, 세리프 디스플레이(Fraunces), 괘선·번호 매김 | `#f7f8f3` / `#20231f` / `#0f5c47` |
| 03 | **Console** | 개발자 다크모드 | Linear 풍 다크, 그리드 배경, 모노스페이스 라벨, 터미널형 프리뷰, 네온 틸 | `#0a0c10` / `#14e0a5` |
| 04 | **Canvas** | 글래스/그라데이션 | 비비드 그라데이션 메시 + 글래스모피즘, 떠 있는 둥근 네비, 컬러풀 모던 SaaS | `#0b0a17` + 블루·바이올렛·민트 |
| 05 | **Blueprint** | 엔터프라이즈 B2B | 블루프린트 그리드, 파이프라인 다이어그램, 비교 표, 보안 섹션, 데이터 밀도 | `#f8fafc` / `#0f172a` / `#1e40af` |

### 01 · Aurora (요청하신 "애플 홈페이지 스타일")
- 한 줄에 한 문장, 큰 헤드라인과 그라데이션 키워드
- 섹션마다 큰 선언형 카피 + 제품 스크린 카드
- 둥근 28px 카드, 부드러운 그림자, hover 부양 효과
- 스크롤 시 콘텐츠 페이드업

### 02 · Studio
- 기존 브랜드 색을 잃지 않으면서 "출판물" 느낌으로 격상
- 세리프 제목 + 산세리프 본문의 대비
- 검토 체인을 ①②③④ 번호 흐름으로 표현

### 03 · Console
- 모델 호출/파이프라인이라는 제품 본질을 "개발 도구" 미학으로 표현
- `step.1 gpt → draft()` 같은 함수형 표기, 상태코드(200 OK) 메타
- 다크 배경에서 결과 카드 가독성 최적화

### 04 · Canvas
- 다중 모델의 "다채로움"을 컬러 그라데이션으로 은유
- 글래스 카드로 깊이감, 둥근 알약형 액션 버튼
- 마케팅 임팩트가 강한 시안

### 05 · Blueprint
- 팀/엔터프라이즈 도입 검토자를 겨냥한 신뢰형 디자인
- 검토 흐름을 설계도(파이프라인 노드)로 시각화
- 기능 비교 표 + 데이터 소유권/보안 섹션 포함

> 시각 언어(색·타이포·레이아웃·모션)만 차용한 예시입니다. 실제 브랜드 자산은 사용하지 않았습니다.

## 공통 설계 원칙 (모든 시안)
- 좌(모델 선택) · 중(입력+결과 카드) · 우(검토 체인 경로/실행 통계) 3패널 워크벤치
- 결과 카드 = 1급 객체: 분기 / 비판 / 개선 / 요약 / 최종 선택 액션
- 모델별 색상 코딩(GPT·Claude·Gemini·Grok), 토큰·지연·예상 비용 메타 표기
- 스트리밍 중 상태(생성 중…) 스켈레톤
- 한국어 우선(현재 ko/en 다국어 제품 성격 반영)

## 기술 메모
- 각 파일은 의존성 없는 단일 HTML (Tailwind Play CDN). 빌드 불필요, 바로 열림.
- 실제 적용 시에는 이 정적 시안을 기준으로 `src/app/page.tsx`,
  `src/components/workbench/*`, `globals.css`의 토큰/클래스를 점진 교체하면 됩니다.

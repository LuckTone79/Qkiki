Qkiki · UI 디자인 컨셉

UI 개선 시안 5종

제품 구성(멀티-AI 비교·검토 체인·분기·프로젝트·프리셋)은 그대로 유지하고, 디자인(UI) 측면만 5가지 방향으로 재해석한 예시입니다. 각 시안은 랜딩페이지와 세부페이지(워크벤치)를 포함합니다. 예시용 목업이며 실제 제품에는 적용되지 않았습니다.

01

Aurora · Apple 홈페이지 스타일

초대형 타이포, 넉넉한 여백, 중앙 정렬 히어로, 스크롤 페이드인, 떠 있는 제품 프리뷰. 미니멀·프리미엄.

랜딩워크벤치
02

Studio · 에디토리얼 미니멀

현재 크림·틸 색감을 계승해 세련화. 세리프(Fraunces) 디스플레이, 괘선·번호 매김, 잡지 같은 정돈된 레이아웃.

랜딩워크벤치
03

Console · 개발자 다크모드

Linear 풍 다크 UI. 그리드 배경, 모노스페이스 라벨, 터미널형 프리뷰, 네온 틸 포인트. 파워유저·기술 지향.

랜딩워크벤치
04

Canvas · 글래스/그라데이션

비비드 그라데이션 메시 + 글래스모피즘 카드. 떠 있는 둥근 네비, 컬러풀하지만 정돈된 모던 SaaS 무드.

랜딩워크벤치
05

Blueprint · 엔터프라이즈

블루프린트 그리드 + 파이프라인 다이어그램, 비교 표, 보안 섹션. 데이터 밀도 높은 신뢰감 있는 B2B 톤.

## 작업 요약
병렬 실행 뒤 생성되는 `AI 결과 차이 비교`가 계속 로딩 상태에 머무는 문제를 수정했다. 비교 요청 effect가 자체 상태 변경으로 즉시 abort 되던 구조를 고쳤고, 비교 요약 생성 모델을 `openai/gpt-5.4`로 고정했다.

## 변경 사항
### 수정된 사항
- 병렬 비교 effect의 의존성 구조를 정리해 비교 요청이 시작 직후 취소되지 않도록 수정.
- `AI 결과 차이 비교` 생성 모델을 `openai/gpt-5.4`로 고정.
- 앱 표시 버전과 루트 버전 파일을 `v1.9.3-20260514`로 갱신.

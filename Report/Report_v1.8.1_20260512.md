## 작업 요약
관리자 쿠폰 관리에서 사용 중인 쿠폰도 비활성화/삭제할 수 있도록 관리자 화면과 API 제약을 수정했다.
관리자가 사용 중 쿠폰을 비활성화/삭제하면 사용자 구독의 쿠폰 혜택이 회수되고, 사용자 쿠폰 화면에 "쿠폰 비활성화" 메시지가 표시되도록 연동했다.

## 변경 사항
### 추가된 기능
- 관리자에 의한 쿠폰 비활성화/삭제 시 사용자 구독 혜택 회수 로직 추가
- 사용자 구독 조회 응답에 쿠폰 상태(`couponStatus`) 추가
- 사용자 계정 쿠폰 화면에 "쿠폰 비활성화" 상태 메시지 표시

### 수정된 사항
- 관리자 쿠폰 목록 UI에서 사용 중 쿠폰에도 비활성화/삭제 버튼 노출
- 관리자 쿠폰 삭제 API의 사용 완료 쿠폰 삭제 제한 제거

### 삭제/제거된 사항
- 없음

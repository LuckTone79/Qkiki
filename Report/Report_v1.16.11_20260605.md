## 작업 요약
워크벤치 입력 패널의 상단 조작 방식을 다시 정리했다. 실행 모드를 드롭다운 대신 `병렬비교`와 `순차검토체인` 개별 버튼으로 노출했고, `질문 복사` 버튼을 같은 줄에 배치했다. 또한 `간단 모드`를 제거하고 항상 고급 설정이 보이도록 통일해, 화면과 실제 실행 payload가 서로 다르게 동작하던 분기를 없앴다.

## 변경 사항
### 추가/개선된 기능
- 입력 패널 헤더에 `병렬비교`, `순차검토체인`, `질문 복사` 버튼을 한 줄에 배치
- 실행 모드 선택을 드롭다운에서 즉시 클릭 가능한 2버튼 방식으로 변경
- 추가 지시, 첨부, 반복/조기종료, 프리셋 저장 영역을 항상 노출하도록 정리

### 수정된 사항
- `simple/advanced` 빌더 분기 제거
- 실행 payload 생성 시 항상 현재 고급 설정 값을 반영하도록 `workbench-run-payload` 단순화
- 관련 테스트를 새 동작 기준으로 갱신
- 앱 표시 버전을 `v1.16.11-20260605`로 업데이트

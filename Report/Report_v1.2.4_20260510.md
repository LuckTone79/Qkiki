## 작업 요약
비로그인 사용자가 랜딩 화면의 `시작하기`를 눌렀을 때 로그인 화면으로 바로 이동하던 동작을 수정했습니다.
이제 같은 기기에서 기존 trial 기록이 있더라도 5회 체험을 모두 소진하지 않았다면 기존 trial 계정 세션을 다시 열고 워크벤치로 진입합니다.

## 변경 사항
### 수정된 사항
- 기존 `TrialAccess` 기록이 있는 경우, 남은 체험 횟수가 있으면 로그인 페이지 대신 trial 세션을 재생성하도록 변경
- 체험 5회를 모두 사용한 경우에는 기존처럼 로그인 유도 경로로 이동하도록 유지
- 애플리케이션 버전을 `v1.2.4-20260510`으로 업데이트

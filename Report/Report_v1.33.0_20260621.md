# 작업 보고서

## 기본 정보
- **버전**: v1.33.0-20260621
- **작업 일시**: 2026-06-21
- **이전 버전**: v1.32.13-20260621
- **프로젝트명**: Yapp (qkiki-workbench)

## 작업 요약
관리자 Users 탭을 사용자 사용량 중심으로 재구성했습니다. 사용자가 실행한 작업의 최신 활동, 작업 수, 크레딧, 토큰, 비용, 최근 작업별 사용량을 상세 화면 클릭 없이 목록에서 바로 볼 수 있고, 최신 사용순/최대 작업순/최대 크레딧 사용순/최대 토큰 사용순 필터링과 정렬을 추가했습니다.

## 변경 사항
### 추가된 기능
- 관리자 사용자 목록 필터: 상태, 역할, 전체 로드.
- 관리자 사용자 목록 정렬: 최신 사용순, 최대 작업순, 최대 크레딧 사용순, 최대 토큰 사용순, 최신 가입순.
- 사용자별 총 작업 수, 완료/진행 중 작업 수, 사용 크레딧, 입력/출력 토큰, 추정 비용 표시.
- 사용자별 최근 작업 사용량을 목록에서 직접 표시.
- `/api/admin/users`와 `/admin/users`가 같은 집계 로직을 공유하도록 서버 헬퍼 추가.

### 수정된 사항
- 작업 사용량 정산 시 `User.lastActiveAt`을 갱신해 관리자 메뉴 최신 사용 정보가 바로 반영되도록 수정.
- 관리자 사용자 목록과 API를 `force-dynamic`, `revalidate = 0`, `Cache-Control: no-store` 흐름으로 정리.
- 사용자 활동 집계 테스트를 추가해 `UsageLog`와 실행 상태가 크레딧/토큰/최신 사용 정렬에 반영되는지 검증.

### 삭제/제거된 사항
- 없음

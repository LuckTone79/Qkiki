# v1.16.1 — 세부페이지 Notion 외관 강화

- 일자: 2026-06-04
- 브랜치: `claude/qkiki-ui-design-concepts-m1CR7`
- 기준: v1.16.0 → **v1.16.1**

## 배경
v1.16.0은 Tailwind 색 토큰만 노션 그레이로 재매핑했는데, 기존 크림/stone 팔레트와
색감이 비슷해 노션 특유의 외관이 시각적으로 드러나지 않았다. 이번 패치는 모든
세부페이지를 감싸는 프레임·공용 헤더에 **노션의 시각적 시그니처**를 직접 적용한다.
랜딩(Tesla)은 그대로 둔다. 기능·구조는 변경 없음.

레퍼런스: `design-concepts/15-notion/workbench.html`

## 적용한 노션 시그니처
1. **세리프 제목 (Lora)** — 노션은 본문 산세리프 + 제목 세리프 대비가 핵심
   - `SectionHeader` 제목 → `font-serif`. 이 컴포넌트는 워크벤치·프로젝트·프로젝트상세·
     세션·프리셋·계정 6개 페이지의 H1으로 쓰여, 한 번의 수정으로 전 페이지 제목이 노션화.
   - 가이드 페이지 히어로 H1 + 모든 섹션 H2/H3 → 세리프
   - 사인인/사인업 제목 → 세리프
2. **⬡ 로고 + 세리프 워드마크** — AppShell, AdminShell, 인증 페이지
3. **이모지 내비게이션** — AppShell 좌측/모바일 nav에 항목별 이모지
   (🧪 워크벤치 · 🗂️ 프로젝트 · 📄 세션 · ⚡ 프리셋 · 👤 계정), 프로젝트 목록 📁
4. **노션식 섹션 라벨** — `uppercase tracking-[0.12em] text-stone-400/500` 로 통일
5. **둥글기·여백 정리** — 패널 `rounded-md`, nav 아이템 `px-2 py-1.5 hover:bg-soft`,
   인증 카드 그림자 제거(노션은 플랫)

## 자동 전파 (v1.16.0 색 토큰 재매핑 위에 적용)
- 모든 `/app/*` 페이지 — AppShell 프레임으로 노션 크롬
- 모든 `/admin/*` 페이지 — AdminShell 프레임으로 노션 크롬
- 워크벤치·프로젝트·세션·프리셋·계정·프로젝트상세 — SectionHeader 세리프 제목
- 사인인·사인업·가이드 — 세리프 + 노션 톤
- 결과 카드·테이블·패널 본문 — 재매핑된 노션 중성색 자동 적용

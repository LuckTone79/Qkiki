## 작업 요약
워크벤치 실행이 `Disabled by administrator` 상태 때문에 막히던 문제를 복구했습니다.
운영 환경변수 키가 이미 존재하는 공급자는 관리자 설정 레코드가 비어 있어도 자동으로 활성 상태로 인식하고 사용할 수 있도록 수정했습니다.

## 변경 사항
### 수정된 사항
- 공급자 가용성 판단 시 환경변수 키가 있는 경우, 관리자 설정 레코드가 없으면 자동 활성 처리
- 첫 실행 시 누락된 `adminProviderConfig`를 자동 생성하여 이후에도 정상 상태 유지
- 관리자 공급자 상태 조회 및 헬스체크가 동일한 기준으로 동작하도록 정리
- 비활성 공급자는 워크벤치에서 체크박스/모델 선택이 비활성화되도록 수정
- 한국어 UI에서도 백엔드 상세 오류 문구를 그대로 표시하도록 수정
- 애플리케이션 버전을 `v1.2.5-20260510`으로 갱신

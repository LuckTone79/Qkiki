## 작업 요약
공유 링크 생성이 브라우저 클립보드 권한 실패에 의해 중단되는 문제를 수정했다.
링크 발급과 클립보드 복사를 분리해, 자동 복사가 차단되어도 공유 링크가 화면에 표시되고 바로 열 수 있도록 구조를 개선했다.

## 변경 사항
### 추가된 기능
- 클립보드 복사 실패를 예외 대신 결과값으로 반환하는 브라우저 클립보드 헬퍼 추가
- 전체 공유 링크 생성 후 URL 표시 및 열기 링크 제공
- 개별 결과 공유 링크 생성 후 URL 표시 및 열기 링크 제공
- 클립보드 차단/미지원 상황에 대한 단위 테스트 추가

### 수정된 사항
- 공유 링크 발급 성공과 자동 복사 성공을 분리
- 자동 복사가 차단되어도 공유 기능이 실패하지 않도록 UI 상태 처리 변경
- 결과 본문 복사 버튼도 클립보드 실패 시 화면 흐름을 깨지 않도록 변경

### 삭제/제거된 사항
- 없음

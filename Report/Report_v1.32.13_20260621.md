## 기본 정보
- **버전**: v1.32.13-20260621

- **이전 버전**: v1.32.12-20260620

## 작업 요약
긴 순차 실행 중 사용자가 브라우저를 나갔다가 돌아와도 현재 실행 상태를 서버에서 다시 동기화하도록 보강했습니다. V2 백그라운드 단계 실행의 토큰/비용 기록이 관리자 대시보드에 누락되는 문제를 수정하고, 과거 누락된 V2 결과도 보조 집계로 반영되게 했습니다.

## 변경 사항
### 추가된 기능
- 페이지 복귀(`pageshow`) 및 탭 재표시(`visibilitychange`) 시 현재 실행/세션 상태 재검증
- V2 단계 완료 시 `ai_requests` 기록 생성
- `ai_requests`가 없는 기존 V2 단계 결과를 관리자 대시보드 사용량에 보조 집계

### 수정된 사항
- 실행 완료 후 usage 이벤트가 없을 때 `/api/usage`를 캐시 우회로 다시 조회
- 관리자 Provider/Model/User 사용량 집계가 V2 단계 결과의 호출 수, 입력 토큰, 출력 토큰, 비용을 포함하도록 수정

### 삭제/제거된 사항
- 없음

## 작업 요약
OpenAI/xAI 호출 포맷 수정 후에도 워크벤치 진입 직후 일부 요청이 500으로 실패하던 문제를 추가로 해결했습니다.
원인은 앱 셸의 여러 링크가 자동 prefetch를 동시에 발생시켜 DB 세션 풀을 빠르게 소진시키는 것이었습니다.

## 변경 사항
### 수정된 사항
- 사이드바 및 하단 네비게이션의 내부 링크에 `prefetch={false}` 적용
- 프로젝트 목록/새 프로젝트/더 보기 링크에도 `prefetch={false}` 적용
- 워크벤치 진입 직후 불필요한 병렬 서버 요청을 줄여 DB 연결 초과 가능성 완화
- 애플리케이션 버전을 `v1.2.7-20260510`으로 갱신

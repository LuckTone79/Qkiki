## 작업 요약
관리자 대화 탭의 모바일 화면 폭 최적화를 개선하고, 각 대화 카드에 소요 크레딧, 총 토큰 수, 추정 금액을 표시하도록 추가했다. 세션별 사용량 집계는 결과 레코드를 우선 사용하고, 결과에 연결되지 않은 ai_requests만 보조 합산해 중복 집계를 피한다.

## 변경 사항
### 추가된 기능
- 관리자 대화 목록에 대화별 소요 크레딧, 총 토큰 수, 추정 금액 표시 추가
- 세션별 결과/요청 사용량 집계 유틸 및 회귀 테스트 추가

### 수정된 사항
- 모바일 관리자 하단 메뉴를 가로 스크롤 대신 4열 2줄 그리드로 최적화
- 대화 카드 제목, 이메일, 사용량 메타 영역이 모바일에서 화면 폭을 밀어내지 않도록 줄바꿈/최소폭 조정
- 앱 표시 버전을 v1.34.0-20260622로 갱신

### 삭제/제거된 사항
- 없음

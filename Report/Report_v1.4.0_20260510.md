# Qkiki v1.4.0-20260510 작업 보고서

## 작업 개요
- 워크벤치 UI와 병렬 비교 흐름을 정리해 요청하신 6개 항목을 반영했습니다.
- 병렬 결과의 차이점을 별도 AI 비교 요약으로 생성하는 서버 경로를 추가했습니다.
- 좌측 사이드바의 탐색성과 공간 활용을 개선했습니다.

## 반영 내용
1. 워크플로우 빌더는 `순차검토체인` 모드에서만 표시되도록 변경했습니다.
2. AI 실행 상태 배지에 모래시계 회전 애니메이션을 추가했습니다.
3. 좌측 `세션` 메뉴 아래에 최근 작업 10개를 자동 표시하고, 초과 시 `...` 링크를 노출하도록 구현했습니다.
4. 좌측 메뉴 접기는 상단 버튼 대신 사이드바 오른쪽 가장자리의 작은 화살표 버튼으로 교체했습니다.
5. 병렬 비교 결과 배치의 기본값을 `2열`로 변경하고, 저장 키도 갱신해 기본값이 바로 반영되도록 조정했습니다.
6. 병렬 상위 결과를 기준으로 모델별 공통점과 차이점을 요약하는 `AI 결과 차이 비교` 패널과 `/api/workbench/compare` API를 추가했습니다.

## 참고
- 병렬 결과 차이 비교는 현재 OpenAI 기본 모델(`gpt-5.5`)을 사용하도록 구현했습니다.
- 세션을 다시 열었을 때도 상위 병렬 결과가 2개 이상이면 비교 요약을 다시 생성하도록 구성했습니다.

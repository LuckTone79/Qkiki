## 작업 요약
워크벤치의 실행 버튼을 모바일과 데스크탑 모두에서 더 크고 눈에 띄는 CTA로 개선했다.
병렬 실행 영역과 순차 검토 체인 영역의 실행 버튼이 같은 스타일을 공유하도록 정리했다.

## 변경 사항
### 추가된 기능
- 실행 버튼 공통 스타일 상수 `runButtonClassName`을 추가했다.
- 실행 버튼에 큰 터치 영역, 오렌지 색상, 그림자, 링, 포커스 상태를 적용했다.

### 수정된 사항
- 병렬/입력 패널 실행 버튼을 기존 검정 소형 버튼에서 오렌지 강조 버튼으로 변경했다.
- 순차 워크플로우 패널 실행 버튼도 동일한 강조 스타일로 변경했다.

### 삭제/제거된 사항
- 없음

# 작업 보고서

## 기본 정보
- **버전**: v1.24.0-20260612
- **작업 일시**: 2026-06-12
- **이전 버전**: v1.23.0-20260612
- **프로젝트명**: Qkiki Multi AI Workbench

## 작업 요약
GPT/Claude의 월 $20 개인 구독제와 비교했을 때 Qkiki의 기존 최저 유료 플랜이 과도하게 높게 보이는 문제를 반영해, 최저 플랜을 월 $11.30으로 낮추고 전체 가격/크레딧 계단을 재설계했습니다. 공개 가격표와 실제 사용 정책이 같은 플랜 카탈로그를 참조하도록 정리했습니다.

## 변경 사항
### 추가된 기능
- 공통 요금제 카탈로그 `src/lib/billing-plans.ts` 추가
- Starter/Pro/Team/Credit Pack의 USD 가격과 크레딧 정책을 한 곳에서 관리

### 수정된 사항
- Starter: $11.30/mo, 700 credits/month, 120 daily credits
- Pro: $29/mo, 2,400 credits/month, 400 daily credits
- Team: $89/mo, 7,500 shared credits/month, 1,300 daily shared credits
- Credit Pack: $39, 2,500 credits
- `/app/pricing` 공개 가격표를 새 USD 가격 구조로 개편
- 실제 사용 정책의 월/일 크레딧과 요청 한도를 새 가격 구조에 맞춰 축소/조정
- 글로벌 수익화 가이드와 구독 크레딧 설계 문서의 기존 $29 최저가 기준을 새 구조로 갱신
- 앱 버전을 `v1.24.0-20260612`로 업데이트

### 삭제/제거된 사항
- 없음

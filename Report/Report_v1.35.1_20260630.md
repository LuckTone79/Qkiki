## 작업 요약
프로젝트 생성 실패 화면과 Yapp 운영 로그를 대조한 결과, 해당 요청은
데이터베이스 오류가 아니라 만료된 인증 세션으로 인한 `401 Unauthorized`였습니다.
한국어 화면이 이 응답을 일반 생성 오류로 표시해 실제 원인을 알 수 없었던 흐름을
로그인 복귀 방식으로 수정했습니다.

## 변경 사항
- 프로젝트 목록 조회와 생성 요청에서 `401` 응답을 별도로 처리합니다.
- 인증이 만료되면 로그인 화면으로 이동하고, 로그인 후 프로젝트 화면으로
  돌아오도록 내부 복귀 경로를 전달합니다.
- 프로젝트 생성 중 만료된 경우 `/app/projects?create=1`로 복귀해 생성 폼을 다시
  표시합니다.
- 서버가 전달한 이동 경로는 내부 절대 경로만 허용해 외부 URL 이동을 차단합니다.
- 인증 이동 규칙에 대한 회귀 테스트 3건을 추가했습니다.

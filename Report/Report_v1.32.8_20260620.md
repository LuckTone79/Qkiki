# 작업 보고서

## 기본 정보
- **버전**: v1.32.8-20260620
- **작업 일시**: 2026-06-20
- **이전 버전**: v1.32.7-20260620
- **프로젝트명**: Yapp 오케스트레이션 워크벤치

## 작업 요약
관리자 인증 쿠키 전환과 관리자 콘솔 모바일 화면 개선을 정리했습니다. 관리자 세션도 새 Yapp 쿠키를 우선 사용하면서 기존 Qkiki 관리자 쿠키를 읽고 정리할 수 있도록 했고, 관리자 목록/테이블 화면은 모바일에서 카드형 표시를 제공합니다.

## 변경 사항
### 추가된 기능
- 관리자 콘솔 모바일 헤더에 언어 선택 컨트롤을 추가했습니다.
- 관리자 대시보드 사용량 표, 피드백 목록, 사용자 사용 로그에 모바일 카드 레이아웃을 추가했습니다.

### 수정된 사항
- 관리자 세션 조회/로그아웃이 새 쿠키와 legacy 쿠키 후보를 모두 처리하도록 수정했습니다.
- 관리자 대화/공급자/피드백/사용자 상세 화면의 긴 텍스트와 버튼 폭을 모바일에 맞게 정리했습니다.
- 관리자 정보 화면의 한국어 문구를 정상 문자열로 복구했습니다.

### 삭제/제거된 사항
- 없음

# Report v1.36.0-20260710 — Genspark 레퍼런스 기반 UI 리디자인 시안 7종

## 요청
사용자가 Genspark 모바일 앱 스크린샷 4장을 레퍼런스로 첨부하고, 현재 Qkiki UI가
"AI가 대충 만든 느낌"이라며 UI를 어떻게 바꿀 수 있는지 **디자인 + 이미지화된
추천 리스트 5개 이상**을 요청.

## 산출물
- `design-concepts/16-genspark-redesign/index.html` — 시안 7종 + 진단 + 적용 로드맵이
  담긴 단일 프레젠테이션 페이지 (순수 HTML/CSS 목업, 라이트/다크 대응, 외부 의존성 없음)
- **제품 코드(`src/`)에는 미적용.** 기존 design-concepts 폴더 관례와 동일하게 시안 전용.
  (버전 표기 규칙 준수를 위한 `VERSION`/`src/lib/version.ts` 버전 문자열 갱신만 포함)

## 레퍼런스에서 추출한 디자인 언어
초대형 라운드(26px) 컴포저 카드 · 스트로크 라인 아이콘 타일 그리드 · 흑백 필(pill)
버튼 위계 · 바텀시트(그래버 + 3타일 + 아이콘 행 리스트) · 4탭 하단 내비 ·
파랑(#2F6BFF) 단일 포인트(선택/업그레이드 전용) · 산세리프 800 헤드라인.

## 시안 리스트
1. **컴포저 허브 홈** — 이중 텍스트영역+점선 첨부 박스를 단일 컴포저 카드로 통합,
   기능(병렬/체인/이미지/프리셋 등)은 도구 타일 그리드로.
2. **데스크톱 아이콘 레일 셸** — 이모지 사이드바 → 64px 라인 아이콘 레일 + 중앙 컴포저.
3. **모델 브랜드 타일** — ProviderSelectorRow 체크박스 → 브랜드 컬러 타일 + 세부 모델 칩.
4. **4탭 내비 + 바텀시트** — 모바일 7탭 → 4탭(홈/세션/프로젝트/더보기), 첨부·프리셋은 시트.
5. **결과 카드 스트림** — ResultCard를 아바타+회색 라운드 카드+아이콘 액션 행으로,
   채택 카드는 검정 테두리 승격.
6. **모노 필 컴포넌트 시스템** — globals.css 토큰 교체(쿨 뉴트럴, 필 위계, 라운드 체계,
   세리프 헤딩 제거). 1~5번의 공통 기반.
7. **플랜 & 사용량 시트** — 실제 요금제(Starter $7.3/Pro $19/Team $59)를 라인아이콘
   혜택 리스트 + 라디오 플랜 카드 + 검정 CTA로.

## 진단 (현재 UI의 문제)
이모지 아이콘 내비 · 폼처럼 보이는 입력부 · 체크박스 관리화면 같은 모델 선택 ·
6px 라운드/테두리 과다 · 포인트 컬러 부재(teal→회색 리매핑).

## 추천 적용 순서
1단계 06(토큰+아이콘) → 2단계 01+02(셸/입력부) → 3단계 03+05(모델선택/결과카드)
→ 4단계 04+07(모바일 시트/과금).

Playwright(Chromium)로 전체 페이지 및 8개 섹션 렌더링 스크린샷 확인 — 레이아웃 깨짐,
줄바꿈 오류(필 버튼 개행, 한국어 단어 분리) 수정 완료.

## 작업 요약

## 변경 사항
### 추가된 기능

- 경고 배너에 포커스를 줄 수 있도록 `tabIndex`와 포커스 링을 추가했다.

### 수정된 사항
- 실행 전 예상 크레딧 부족 분기에서 에러 메시지 표시 후 즉시 경고 위치로 이동한다.
- 서버 사용량 제한 응답 처리에서도 같은 경고 위치 이동을 수행한다.

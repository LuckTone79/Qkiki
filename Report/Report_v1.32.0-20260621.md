# 작업 보고서 — v1.32.0-20260621

## 개요
관리자 메뉴 개선 요청에 따라 사용자 사용 통계의 실시간 반영, 사용자 목록 필터/정렬,
작업별 크레딧·토큰 사용 내역 노출을 구현했다. 리브랜딩(Yapp, yapp.wideget.net)은
기존 코드에 이미 반영되어 있어 신규 화면도 동일한 명칭을 유지했다.

## 변경 사항

### 1. 사용 내용이 관리자 메뉴에 즉시 반영되지 않던 문제 (요청 1)
- 원인: `lastActiveAt`(마지막 활동)이 로그인 시점(`createAuthSession`)에만 갱신되어
  사용자가 실제 작업(AI 요청)을 해도 관리자 화면에 반영되지 않았다.
- 조치: `recordUsageSuccess`(`src/lib/usage-policy.ts`) 트랜잭션에서 사용 기록과 함께
  `user.lastActiveAt`을 현재 시각으로 갱신하도록 추가.
- 관리자 사용자 목록/대시보드는 이미 `force-dynamic`이라 매 요청마다 최신 데이터를 조회한다.
  추가로 사용자 목록에 **새로고침 버튼**을 두어 즉시 재조회가 가능하도록 했다.

### 2. 사용자 탭 필터/정렬 기능 추가 (요청 2)
- `src/lib/admin-users.ts` 신규: 사용자 목록을 사용량 집계와 함께 로드하고 정렬하는 공용 헬퍼.
- 정렬 옵션: 최신 사용순(기본), 최신 가입순, 최대 작업순, 최대 모델 호출순,
  최대 크레딧 사용순, 최대 예상 비용순, 최대 토큰 사용순.
- 검색(이메일/이름) + 정렬 드롭다운 UI를 사용자 목록 상단에 추가.
- `GET /api/admin/users`도 동일 헬퍼/정렬 파라미터를 사용하도록 통일.

### 3. 작업별 크레딧·토큰 사용 내역을 클릭 없이 노출 (요청 3)
- 사용자 목록 테이블/모바일 카드에 **작업 수, 모델 호출 수, 크레딧, 토큰(입력/출력),
  예상 비용** 컬럼을 추가하여 상세 진입 없이 확인 가능.
- 목록 상단에 표시된 사용자 기준 합계 요약 카드(사용자 수/작업/호출/크레딧/비용) 추가.

### 추가 개선 (직관적 통계)
- 사용자 상세 화면 상단에 **사용량 요약 카드**(작업/호출/크레딧/토큰/비용)와
  **크레딧 지갑**(유료/보너스/사용) 정보를 추가.
- 상세의 대화 목록을 표 형태로 바꿔 **작업별 호출 수·토큰·비용**을 함께 표시.
- 마지막 활동 시각을 상세 헤더에 노출.

- **이전 버전**: v1.16.9-20260605

## 작업 요약
세션 화면과 작업 입력 화면에서 사용자가 이전에 작성한 요청/질문 내용을 바로 복사할 수 있도록 복사 버튼을 추가했다. 브라우저가 클립보드 쓰기를 막는 경우에도 실패 원인을 바로 알 수 있게 언어별 안내 문구를 함께 정리했다.

## 변경 사항
### 추가된 기능
- 세션 목록 카드에 원본 입력 복사 버튼 추가
- 작업 입력 패널 상단에 현재 질문 복사 버튼 추가
- 원본 입력 복사 성공/실패 상태를 언어별로 보여주는 공용 메시지 헬퍼 추가
- 세션 입력 복사 안내 문구 테스트 추가

### 수정된 사항
- 클립보드 유틸 타입을 외부에서도 재사용할 수 있게 export 정리
- 사용자 공지 영역에서 오류 배너와 별도로 복사 결과 안내를 표시하도록 조정
- 앱 표시 버전을 `v1.16.10-20260605`로 업데이트

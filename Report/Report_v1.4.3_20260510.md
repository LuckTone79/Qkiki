## 작업 요약
사용자 요청에 맞춰 두 가지를 적용했습니다.

1. 모든 AI 공급자의 기본 타임아웃을 300초로 통일
2. 병렬 AI 실행 시 먼저 완료된 모델의 결과가 먼저 보이도록 결과 정렬 방식 수정

## 변경 사항
### 수정한 내용
- OpenAI, Anthropic, Google, xAI의 기본 타임아웃을 모두 300초로 통일
- 기존 legacy 기본값(60, 90, 180초)으로 남아 있는 공급자 설정도 런타임에서 300초로 자동 승격되도록 수정
- 병렬 실행 결과 카드의 정렬 기준을 생성 시각이 아니라 완료 시각(`updatedAt`) 우선으로 변경
- 병렬 비교 요약 영역의 모델 배지 정렬도 완료 시각 기준으로 변경
- 분기 결과는 부모 결과 아래 기존 생성 순서를 유지하도록 정렬 안정성 유지

- 세리프(Lora) 헤딩 폐지: `--font-serif`를 산세리프 스택으로 매핑

### 2. 아이콘 시스템 (`src/components/ui/icons.tsx` 신규)
- 스트로크 1.8px 라인 아이콘 세트(홈/폴더/문서/프리셋/가이드/계정/피드백/더보기 등)
- **공식 AI 로고 마크**: OpenAI 노트, Anthropic A, Gemini 그라디언트 스파크, xAI
- `ProviderLogoTile`: 앱 아이콘 관례 배경(검정/크림 #EEECE2/화이트)의 로고 타일

### 3. AppShell (`src/components/AppShell.tsx`) — 시안 02+04
- 이모지 내비(🧪🗂️📄⚡📘👤💬) → 라인 아이콘 전면 교체
- 브랜드: ⬡ Yapp(세리프) → **검정 스파크 마크 타일 + Qkiki**(산세리프 800)
- 모바일 하단 바: 7탭 → **4탭(홈·프로젝트·세션·더보기)**, 나머지(프리셋/가이드/계정/
  피드백)는 **더보기 바텀시트**(그래버+아이콘 행+버전+로그아웃)로 이동
- sign-in/sign-up 페이지 브랜드 헤더도 동일 마크로 교체

### 4. 모델 선택 (`src/components/workbench/ProviderSelectorRow.tsx`) — 시안 03
- 체크박스 행 → **브랜드 로고 타일 카드**: 탭하여 켜고 끄기, 선택 시 검정 테두리
  + 체크 배지, 비활성 시 상태 문구/저채도
- 세부 모델은 **필 칩 다중 선택**(선택=검정 채움), 추천 모델은 블루 도트
  (세부 특성은 title 툴팁으로 이동해 노이즈 제거). props/동작은 기존과 동일

### 5. 워크벤치 컴포저·결과 카드 — 시안 01+05
- 입력 패널: rounded-3xl 카드 + 소프트 섀도, 병렬/체인 **세그먼트 필**, 이미지 생성·
  질문 복사·파일 첨부 **필 버튼**, 텍스트영역 rounded-2xl, **검정 Run 필**
- ResultCard: 브랜드 배지 → **공식 로고 원형 아바타 + 모델명**, 최종결과 카드는
  검정 테두리+검정 배지로 승격, 액션 버튼 필화, 카드 rounded-2xl
- 워크벤치 패널 12곳 rounded-lg → rounded-2xl 통일, 라벨의 🖼 이모지 제거

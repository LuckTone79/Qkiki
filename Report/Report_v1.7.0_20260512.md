## 작업 요약
Qkiki 프로그램의 공개 가이드북 페이지(`/guide`)를 제작했습니다. 다른 AI 플랫폼과의 차이점을 강조하고, 처음 사용하는 사용자를 위한 8단계 스텝 바이 스텝 매뉴얼을 포함합니다. 제작 후 2회 검증(콘텐츠 정확성 + UX/접근성)을 수행하여 발견된 9개 이슈를 모두 수정했습니다.

## 변경 사항

### 추가된 기능
- **가이드북 페이지** (`/guide`): 공개 접근 가능한 가이드북 상세 페이지
  - 히어로 섹션: Qkiki 소개 및 CTA
  - 차별점 6가지: 병렬 비교, 순차 검토 체인, 분기, 프로젝트, 프리셋, 첨부 파일
  - 8단계 사용 가이드: 타임라인 UI로 시각적 안내
  - 기능 요약 테이블: 지원 모델 17개, 워크플로우 모드, 보안 등
  - FAQ 5개: 아코디언 형태, 접근성 보장
  - 하단 CTA: 회원가입 유도
  - 다국어 지원: 영어/한국어 완전 대응
- **랜딩 페이지 가이드북 링크**: 메인 페이지에 "가이드북" 버튼 추가 (teal 스타일)

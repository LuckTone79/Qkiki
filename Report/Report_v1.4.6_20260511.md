## 작업 요약
로컬 비밀번호 재설정 스크립트가 Prisma 클라이언트 초기화 단계에서 실패하는 문제를 수정했다.
스크립트의 DB 갱신 경로를 `pg` 직접 연결 방식으로 변경해 운영 DB에서 비밀번호와 세션을 안정적으로 갱신할 수 있게 했다.

- 트랜잭션 기반으로 사용자 비밀번호 갱신 및 세션 삭제 처리 유지

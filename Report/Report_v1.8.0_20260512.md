## 작업 요약
가이드북 페이지(`/guide`)에 병렬 비교 및 순차 검토 체인 두 가지 핵심 기능의 상세 비주얼 튜토리얼 섹션을 추가했습니다. 초등학생도 따라할 수 있을 정도의 쉬운 단계별 안내와 실제 화면을 시뮬레이션하는 모의 UI를 포함합니다. 불필요한 FAQ 항목(API 키 관련)을 삭제했습니다.

## 변경 사항

### 추가된 기능
- **병렬 비교 상세 튜토리얼 섹션**: 8단계 따라하기 가이드 + 모의 워크벤치 화면 미리보기
  - 모드 선택, 모델 토글, 질문 입력, 출력 스타일, 파일 첨부, 실행, 대기, 결과 비교
  - 모의 화면: 윈도우 크롬, 모드 셀렉터, 체크박스 토글, 입력 영역, 실행 버튼, 결과 카드(GPT/Claude/Gemini) 시각화
- **순차 검토 체인 상세 튜토리얼 섹션**: 8단계 따라하기 가이드 + 모의 워크벤치 화면 미리보기
  - 모드 선택, 기본 체인 확인, 단계 커스터마이즈, 추가/삭제, 질문 입력, 실행, 체인 실행 과정, 결과 확인
  - 모의 화면: 체인 다이어그램(GPT→Grok→Gemini), 실행 진행 표시(Step 1 ✓ → Step 2 ✓ → Step 3 ⏳)
- 모든 튜토리얼 내용 **영어/한국어 완전 대응**

### 수정된 사항
- `src/app/guide/page.tsx`: 가이드북에 2개 튜토리얼 섹션 추가, FAQ에서 API 키 항목 삭제 (EN/KO 모두)

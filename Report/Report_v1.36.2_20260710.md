# Report v1.36.2-20260710 — 시안 페이지 모바일 레이아웃 깨짐 수정

## 문제 (사용자 제보)
시안 아티팩트를 휴대폰(약 390px 뷰포트)에서 열면 데스크톱 목업 프레임이 깨짐:
- 880px 설계의 데스크톱 프레임(.desk)이 94vw로 줄어들며 내부 요소가 재배치(reflow)됨
- 4열 모델 타일이 세로로 길게 찌그러지고, 텍스트가 글자 단위로 줄바꿈, 하단 실행 바가 잘림
- 660px 컴포넌트 보드(.board)도 동일 위험

## 원인
목업 프레임을 반응형 폭(min(880px, 94vw))으로 두어, 좁은 화면에서 축소가 아니라
내부 레이아웃 붕괴가 일어남. 목업은 문서 내 "그림"이므로 재배치가 아니라
비율 축소(scale)되어야 함.

## 수정 (`design-concepts/16-genspark-redesign/index.html`)
- `.desk`(880px)·`.board`(660px)를 고정 설계폭으로 변경
- 인라인 스크립트 추가: 각 프레임을 `.fit` 래퍼로 감싸고, 컨테이너 폭이 설계폭보다
  좁으면 `transform: scale()`로 통째로 축소 + 래퍼에 축소된 크기 반영(리사이즈 대응)
- 페이지 버전 라벨 v1.36.2 갱신

## 검증
Playwright 390×844(모바일) 뷰포트에서 전 섹션 렌더링:
- `document.scrollWidth` 초과분 0px (가로 오버플로 없음)
- 시안 02·03 데스크톱 프레임, 시안 06 보드가 비율 그대로 축소 렌더 확인

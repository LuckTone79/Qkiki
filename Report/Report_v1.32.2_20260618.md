# Report v1.32.2 - 2026-06-18

## 변경 요약

랜딩 페이지 히어로의 "가이드북" 버튼 가시성 개선.

## 배경

히어로 섹션의 "가이드북" 링크가 배경 없는 반투명 텍스트 링크(`text-white/85` + 밑줄 hover)였음. 복잡한 배경 이미지 위에서 거의 보이지 않아 사용자가 인지하기 어려웠음.

## 변경 내용

### `src/app/page.tsx` (히어로 섹션, line 107)

- 텍스트 링크 → 알약형(rounded-full) 버튼으로 변경
- 책 아이콘(SVG) 추가로 가이드북임을 직관적으로 표시
- 반투명 흰 배경(`bg-white/15`) + 흰 테두리 + `backdrop-blur` + 그림자로 어떤 배경에서도 대비 확보
- hover 시 흰 배경 + 어두운 글자로 반전되어 명확한 상호작용 피드백
- 상단 여백 `mt-3` → `mt-5`로 본문과 시각적 분리

※ 푸터 섹션의 가이드북 버튼(밝은 배경)은 이미 버튼 형태라 변경하지 않음.

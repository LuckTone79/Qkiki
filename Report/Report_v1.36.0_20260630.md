## 작업 요약

애플리케이션 언어에 일본어와 스페인어를 추가했다. 공용 페이지, 인증, 사용자·관리자 메뉴, 워크벤치, 프로젝트, 세션, 프리셋, 피드백, 공유 결과, 가이드북의 기존 영어/한국어 전용 분기를 4개 언어 구조로 전환했다.

## 변경 사항

### 추가 기능

- 언어 선택기에 `日本語`와 `Español` 추가
- 저장된 `ja`, `es` 언어 설정 복원 및 HTML `lang` 반영
- 일본어·스페인어 UI 번역 카탈로그와 공용 로케일 헬퍼 추가
- 가이드북 전체 구조를 일본어·스페인어로 제공
- 스페인어 AI 출력 언어와 앱 언어별 기본 출력 언어 연동 추가

### 수정 사항

- 사용자·관리자 화면의 영어/한국어 이진 분기를 4개 언어 선택으로 변경
- 워크벤치 작업·상태·결과·공유·반복 실행 문구 현지화
- 프로젝트·세션·프리셋·계정·결제·피드백 화면 문구 현지화
- 제품 용어 수동 보정: 워크벤치, 로그아웃, 실행, 닫기, 로딩/저장 상태 등

### 검증

- 지원 언어, 저장값 정규화, 메뉴 핵심 번역 단위 테스트
- 모든 언어 선택기의 4개 옵션 정적 검사
- 가이드북 `en/ko/ja/es` 구조 검사
- 사용자 화면의 `language === "ko"` 이진 분기 제거 검사
- 스페인어 출력 언어 API 스키마 검사
- TypeScript, ESLint, 전체 테스트 198개, 프로덕션 빌드 통과
- 실제 브라우저에서 일본어·스페인어 랜딩/가이드북, HTML `lang`, 콘솔 오류·오버레이 없음 확인
- 프로덕션 배포 후 Yapp 도메인 라이브 검증 수행

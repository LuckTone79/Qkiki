# 작업 보고서
## 기본 정보
- **버전**: v1.4.5-20260511
- **작업 일시**: 2026-05-11
- **이전 버전**: v1.4.4-20260510
- **프로젝트명**: qkiki-workbench

## 작업 요약
운영자 계정 비밀번호를 채팅에 노출하지 않고 로컬 터미널에서 안전하게 재설정할 수 있는 스크립트를 추가했다.
스크립트는 사용자 조회, bcrypt 해시 생성, 비밀번호 갱신, 기존 로그인 세션 정리까지 한 번에 수행한다.

## 변경 사항
### 추가된 기능
- `scripts/reset-password.mjs` 추가
- 숨김 입력 방식의 비밀번호 프롬프트 추가
- 비밀번호 변경 후 기존 사용자/관리자 세션 삭제 처리 추가
- `npm run user:reset-password -- --email ...` 실행 스크립트 추가

### 수정된 사항
- 앱 버전 상수 업데이트
- 루트 `VERSION` 파일 업데이트

### 삭제/제거된 사항
- 없음

## 변경된 주요 파일
| 파일 경로 | 변경 유형 | 설명 |
|----------|---------|------|
| scripts/reset-password.mjs | 추가 | 로컬 터미널용 비밀번호 재설정 스크립트 |
| package.json | 수정 | 비밀번호 재설정 npm 스크립트 등록 |
| VERSION | 수정 | 프로젝트 버전 갱신 |
| src/lib/version.ts | 수정 | 앱 노출 버전 갱신 |

# Report v1.36.0 - 2026-06-18

## 변경 요약

랜딩 페이지 "순차 검토 체인" 섹션에 **반복 루프 설정** 시각화 추가.

## 배경

순차 검토 체인의 핵심 강점 중 하나는 특정 단계 구간을 반복 실행해 품질이 수렴할
때까지 자동으로 다듬는 "반복 루프" 기능(`src/lib/ai/workflow-control.ts`,
워크벤치 "반복 설정")인데, 랜딩 페이지 설명에는 이 내용이 빠져 있었음.

## 변경 내용 (`src/app/page.tsx`)

### 순차 검토 체인 섹션 — 반복 루프 하이라이트 블록 추가
- 기존 체인 흐름(초안→비판→개선→최종) + 자동 전달 캡션 아래에 신규 카드 추가
- teal 강조 테두리 + 순환(loop) 아이콘으로 "반복 루프 설정" 라벨 표시
- 시각 예시: `Grok·비판` ⟲×3 `Gemini·개선` — 비판↔개선 구간을 3회 반복하는 모습
- 설명 캡션: "원하는 단계 구간(시작~종료)을 반복 횟수만큼 자동으로 돌려 품질이
  수렴할 때까지 다듬습니다. 최대 10개 구간 · 총 50단계까지."
  (실제 제약: `MAX_REPEAT_BLOCKS=10`, `MAX_TOTAL_SEQUENTIAL_STEPS=50` 반영)
